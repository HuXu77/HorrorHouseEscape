# Horror House Escape 🏚️

Welcome to the **Horror House Escape** repository! This is a multiplayer horror and puzzle game being built on Roblox. 

In this game, players are trapped in a multi-story house and must work together to find clues, solve puzzles, and escape—all while avoiding an intelligent and terrifying monster that stalks the halls.

## 🛠 Project Structure

- **Engine:** Roblox Studio
- **Language:** Luau
- **Repository:** You are here! We use GitHub to track everything from scripts and `.rbxl` files to project planning.

## 📋 Kanban Board & Issues

We actively track our progress, brainstorming, and bugs using GitHub Projects.

- **Project Board:** [Horror House Escape Kanban](https://github.com/users/HuXu77/projects/1)
- **Issues:** See the [Issues tab](https://github.com/HuXu77/HorrorHouseEscape/issues) for specific tasks.

### Board Columns Explained:
- **`Backlog`**: Brainstorming and future ideas.
- **`To Do (Studio / Human)`**: Physical map building, UI layout, audio uploading. (No PRs, done directly in Studio).
- **`To Do (Scripts / AI)`**: Logic, data saving, pathfinding. (Requires PRs/Commits).
- **`In Progress`**: What a developer or the AI is actively working on.
- **`Review / Testing`**: Code is written but needs to be play-tested in Studio before merging.
- **`Done`**: Merged and verified in the live Studio session.
